# 说明

## 简介

- 1.找到项目中人员协作的关系，找到朋友圈中最活跃、独立的人
- 2.sig maintainer扩展：扩展谁


# 展现

- 节点：
  - 名称：人名
  - 大小：事件个数 
- 边：
  - 在同一个issue中出现的人之间相连接
  - 在同一个pr中出现的人之间相连接
 

过滤条件
- sig
- 领域


### 观察

- 链接度（比如：单打独斗不能胜任）
- 圈层
- 事件大小
